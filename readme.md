# StellarUI

Minimal space-themed landing UI with animated canvas background.

## Preview

![Preview](./assets/readme/stellar-ui-desktop-view.png)
![Preview](./assets/readme/stellar-ui-mobile.gif)

## Stack

- HTML
- CSS
- Vanilla JS (Canvas)

## Features

- Starfield animation (canvas)
- Responsive layout
- Mobile sidebar (accessible)
- Custom cursors

## Run

Open `index.html`

## Notes

No build tools. No dependencies.
